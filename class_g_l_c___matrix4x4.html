Detailed Description

GLC_Matrix4x4 is a 4 dimensions Matrix.

GLC_Matrix4x4 is used to represent 3d homogeneous transformation in 3d space
GLC_Matrix4x4 is a row first matrix compatible with OpenGL Matrix
